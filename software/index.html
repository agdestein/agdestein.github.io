Software

Some open source software projects I have been working on:

  • IncompressibleNavierStokes.jl (Julia): Incompressible fluid flow simulator in 2D and 3D

  • DiscreteFiltering.jl (Julia): Embedded learning of discretely filtered convection operators

  • SpinDoctor (MATLAB) and SpinDoctor.jl (Julia): Finite element diffusion MRI simulator for articificial and real neuron geometries in 2D and 3D

  • Gemseo (Python): Generic Engine for Multidisciplinary Scenarios, Exploration and Optimization

  • Artery.FE (Python): 1D blood flow equation solver based on FEniCS

See also my GitHub page.
